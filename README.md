# Advent Of Code 2020

A Rust implementation of the 2020 edition of [Advent of Code](https://adventofcode.com/2020/).

## Build

```sh
cargo build
```

## Test

```sh
cargo test
```

## Run

### Run all

```sh
cargo run
```

### Run a specific day

```sh
cargo run -- --day 1
```
